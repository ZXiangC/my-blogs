### （2）MybatisX 插件

- 第一步：File -> Settings -> Plugins -> Browse Repositories，输入 mybatisx 搜索并安装。
- 第二步：重启。

功能 ：
   Java 与 XML 调回跳转
   Mapper 方法自动生成 XML
