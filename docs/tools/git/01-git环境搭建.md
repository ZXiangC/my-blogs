### 三 ssh公钥和私钥

#### 3.1 本地使用命令生成公钥和私钥

使用GitBash 输入如下命令

```java
ssh-keygen -t rsa
```

本机公钥和私钥生成位置

```java
C:\Users\祥子\.ssh
```
